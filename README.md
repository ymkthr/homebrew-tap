# Homebrew Tap

Personal Homebrew tap by [@ymkthr](https://github.com/ymkthr).

## Usage

```bash
brew tap ymkthr/tap
```

## Available Casks

### AgentLimits

macOS menu bar app and widgets for Codex/Claude Code usage limits and ccusage.

```bash
brew install --cask ymkthr/tap/agentlimits
```

- Homepage: https://github.com/Nihondo/AgentLimits
- Requires: macOS Sonoma or later
